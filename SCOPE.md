# Scope of the 0Din Bug Bounty Program

This document defines the scope of the 0Din bug bounty program.

## In-Scope Systems
- OpenAI APIs (ChatGPT, GPT-4, DALL-E)
- Anthropic Claude
- Google Gemini
- Meta Llama
- Azure OpenAI
- Other AI/ML systems as specified

## Out-of-Scope Systems
- General website bugs (e.g., XSS on the GitHub repository itself)
- Social engineering
- Physical attacks

Please refer to [SECURITY.md](SECURITY.md) for reporting guidelines.
